## How to Run Code Locally

1. Ensure you have Node.js installed (version 14 or higher).
2. Install TypeScript globally: `npm install -g typescript`
3. For each level, navigate to the directory, e.g., `cd level-1`
4. Compile the TypeScript file: `tsc easy_1.ts`
5. Run the compiled JavaScript: `node easy_1.js`

Alternatively, use ts-node for direct execution: `npx ts-node easy_1.ts`
